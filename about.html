About This Website

This website is a school activity purposes created to demonstrate the basic structure of a multi-page website using HTML and CSS. It focuses on simplicity, clarity, and ease of navigation.

Mission

Our mission is to provide a simple and organized website layout that helps students understand the fundamentals of web design.

Vision

Our vision is to inspire learners to build clean, functional, and user-friendly websites using basic web technologies.
